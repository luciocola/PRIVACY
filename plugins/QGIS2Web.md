# Privacy Statement for qgis2web

## Overview
This privacy statement describes how qgis2web handles user data and privacy. qgis2web is a QGIS plugin that exports QGIS projects to web maps (OpenLayers, Leaflet).

## Data Collection
qgis2web does not collect any user data:
- No usage statistics are collected
- No telemetry is sent to external servers
- No user information is transmitted

## Data Usage
Not applicable - no data is collected.

## Data Storage
- **Location**: All output files are stored locally on the user's machine
- **Duration**: Files persist until manually deleted by the user
- No data is stored on external servers by the plugin itself

## Data Sharing
qgis2web does not share data with third parties. However:
- The exported web maps may include links to external libraries (OpenLayers, Leaflet) if using CDN
- If users choose to deploy the generated web maps online, standard web server logging may apply
- Users are responsible for the privacy implications of publishing their generated web maps

## User Rights
Users have full control over:
- All generated output files
- Whether to publish maps online
- What data to include in exported maps
