# Privacy Statement for QuickMapServices

## Overview
This privacy statement describes how QuickMapServices handles user data and privacy. QuickMapServices is a QGIS plugin that provides easy access to basemap services.

## Data Collection
QuickMapServices collects the following types of data:
- Plugin usage statistics (optional, can be disabled)
- Error reports (only if user chooses to submit)

## Data Usage
The collected data is used for:
- Improving plugin functionality and user experience
- Debugging and fixing errors
- Understanding which basemap services are most popular

## Data Storage
- **Location**: Data is stored on the user's local machine
- **Duration**: Usage statistics are temporary and not transmitted unless explicitly enabled
- Configuration files are stored in the QGIS user profile directory

## Data Sharing
QuickMapServices connects to third-party basemap services:
- OpenStreetMap
- Google Maps (if enabled)
- Bing Maps (if enabled)
- Other configured tile services

When using these services, your IP address and map tile requests are sent to the respective service providers. Please refer to each service provider's privacy policy for information on how they handle this data.

## User Rights
Users have the following rights regarding their data:
- Disable usage statistics collection in plugin settings
- Delete local configuration files
- Choose which basemap services to use

## Changes to This Privacy Statement
This privacy statement may be updated from time to time. The current version is dated January 2026.

## Compliance
This plugin respects user privacy and:
- Does not collect personal information without consent
- Allows users to opt-out of data collection
- Stores data locally by default
