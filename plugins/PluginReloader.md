# Privacy Statement for Plugin Reloader

## Overview
This privacy statement describes how Plugin Reloader handles user data and privacy. Plugin Reloader is a QGIS plugin that allows developers to reload Python plugins during development.

## Data Collection
Plugin Reloader does not collect any user data:
- No usage statistics
- No telemetry
- No personal information

## Data Usage
Not applicable - no data is collected.

## Data Storage
- **Location**: Plugin configuration is stored locally in QGIS settings
- **Duration**: Settings persist until manually cleared or QGIS is reinstalled
- No data is transmitted to external servers

## Data Sharing
Plugin Reloader does not share any data with third parties.

## User Rights
Users have full control over:
- Local plugin settings
- Which plugins to reload
- All data stored on their local machine
