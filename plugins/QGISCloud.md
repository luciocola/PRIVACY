# Privacy Statement for QGIS Cloud

## Overview
This privacy statement describes how QGIS Cloud plugin handles user data and privacy. QGIS Cloud is a plugin that allows users to publish QGIS projects to the cloud.

## Data Collection
QGIS Cloud collects the following types of data:
- User account information (username, email, password)
- QGIS project data uploaded to the cloud
- Database content synchronized with the cloud
- Usage logs and access statistics

## Data Usage
The collected data is used for:
- Providing cloud hosting and publishing services
- User authentication and authorization
- Service improvement and maintenance
- Billing and subscription management

## Data Storage
- **Location**: Data is stored on QGIS Cloud servers (European Union)
- **Duration**: Data is retained while the account is active
- Users can delete their data by closing their account

## Data Sharing
QGIS Cloud may share data with:
- Cloud infrastructure providers for hosting services
- Payment processors for subscription billing
- No data is sold to third parties

Published maps are publicly accessible if shared with a public URL.

## User Rights
Users have the following rights regarding their data:
- Access their account data and projects
- Export their data
- Delete their account and all associated data
- Modify privacy settings for published maps
- Request data corrections

## Contact Information
For privacy-related questions or concerns, please contact:
- **Service Provider**: Sourcepole AG
- **Email**: qgiscloud@sourcepole.ch
- **Website**: https://qgiscloud.com/
- **Privacy Policy**: https://qgiscloud.com/en/pages/privacy

## Changes to This Privacy Statement
This privacy statement may be updated from time to time. The current version is dated January 2026.

## Compliance
QGIS Cloud complies with:
- GDPR (General Data Protection Regulation)
- Swiss Federal Data Protection Act
- EU-US Privacy Shield principles
